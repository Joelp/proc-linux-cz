Linux na serveru

Linux na serveru vyrostl a právě tam má do dnes silné místo. Kvalitní Linuxové jádro vycházející z UNIX-ových standardů společně s technologiemi software s otevřeným nebo svobodným zdrojovým kódem tvoří stabilní celek, který dnes dokáže obstát i v kritických serverových aplikacích.

Linux jako kernel

POSIX

Linuxový kernel, který dal jméno celémy systému, napsaný původně Linusem Torvaldsem je dnes vyvíjen stovkami vývojářů z celého světa, pod jejichž pečlivým dohledem naplňuje specifikace POSIX a "single UNIX specification".

Multitasking, VMM...

Kernel dnes disponuje všemi prvky, které byste očekávali od moderního UNIX-ového systému jako je multitasking, správa virtuální paměti, sdílené knihovny, rozprostření požadavků, správa přidělování zdrojů, podpora síťových technologií IPv4 i IPv6 a další.

Síla 64-bitů a 15-ti architektur

Přestože byl původně napsán pro 32-bitovou architekturu x86, dnes je možné ho používat na množství jiných architektur včetně plně 64-bitových - Compaq Alpha AXP, Sun SPARC a UltraSPARC, Motorola 68000, PowerPC, ARM, Hitachi SuperH, IBM S/390, MIPS, HP PA-RISC, Intel IA-64, DEC VAX, AMD x86-64, AXIS CRIS a Renesas M32R. Sílu 64-bitů již dnes oceňuje množství společností využívající Linux.

Linux jako operační systém

Open a Free software

Linux jako operační systém je celek tvořený Linuxovým kernelem a souborem aplikací zpravidla s otevřeným (Open Source - OSS) nebo svobodným zdrojovým kódem (FSF). Vzniká v mnoha variacích, od těch určených pro osobní počítače až po serverové, které jsou optimalizovány pro nasazení ve firemních aplikacích, ale umožňují dosáhnout na tyto služby i běžným uživatelům.

Server s velkým S

Linux může být použit pro všechny dnes důležité síťové úkoly jako webové servery, bezpečnostní síťové prvky (firewally) a směrovače (routery), poštovní (mail) a antivirové servery včetně ochrany proti nevyžádané poště, souborové, tiskové a autentizacní servery včetně služeb pro platformu MS Windows, databázové servery, komunikační servery (jabber), herní servery, aplikační servery pro intranetové i extranetové aplikace (ERP, GroupWare, CMS, RCS atd), proxy servery, streamingové servery atd. Silnou stránkou je i použitelnost ve víceprocesorových systémech a schopnost sdružování (clusterování) Linuxových systémů, které umožňují využívat Linux pro náročné aplikace ať už vědeckého nebo obchodního charakteru.

Výhodou OSS a FSF řešení je kompatibilita s veřejnými standardy a většinou možnost propojit je i s proprietárními službami. Proprietární řešení naopak často standardy nepodporuje.

Webové služby

Linuxový webový server HTTP většinou postavený na Apache (nicméně je možné použít i jiné specializované HTTP servery) dnes stojí v pozadí majority webových stránek na které narazíte kdekoli na Internetu. Podpora pro silné zabezpečení SSL a standardů pro tvorbu dynamického obsahu CGI/ASP v podobě modulů pro PHP, Python, Perl a dalších dnes předurčuje využívání Linuxových serverů ve všech odvětvích. WebDAV (Web-based Distributed Authoring and Versioning) je novou technologií umožňujíci kolaborativní spolupráci rozprostřených týmů a již dnes si její sílu můžete na Linuxu vyzkoušet.

Firewally a routery

Firewally, směrovače a další prvky nezbytné pro běh moderních sítí využívají přirozených schopností Linuxu umožňujících filtrovat a směrovat provoz na velkém množství síťových technologií. Mezi nepostradatelné služby dnes patří bezpečná komunikace prostřednictvím SSH (OpenSSH), IPsec nebo vytváření virtuálních privátních sítí VPN. Linuxové síťové služby zde zdaleko nekončí a nabízí např. DNS (Domain Name Services - BIND), DHCP (Dynamic Host Configuration Protocol) nebo NTP (Network Time Protocol) a další, vše pro IPv4 i IPv6. Inteligentní směrování provozu je zajištěno prostřednictím routovacího software Quagga (OSPFv2, OSPFv3, RIP v1 a v2, RIPng a BGP-4), bezpečností prověrky sítí pak lze provádět prostřednictvím software Nessus nebo Kismet - pro analýzu bezdrátových sítí.

Mailové, antivirové a antispamové služby

Jednou z nejrozšířenějších oblastí nasazení Linuxu jsou služby elektronické pošty. Linuxové servery dnes spravují, doručují a kontrolují gigantické množství dat elektronických komunikací. Využívají k tomu software Mail Transport Agentů (MTA) jako Sendmail nebo Postfix, antivirové kontroly programy ClamAV a Amavis, identifikace nevyžádané pošty (SPAM) SpamAssassin. Koncovým uživatelům pak servírují poštu přes protokol POP3 nebo IMAP (Dovecot, Cyrus Imapd) včetně jejich zabezpečených verzí a přijímají ji od nich protokolem SMTP. Důležitou složkou elektronické komunikace jsou do dnes konference, jejichž značná část je provozována na systému Mailman.

Souborové, tiskové a autentizační služby

Sdílení souborů, tiskáren a zdrojů obecně, připojování vzdálených svazků nebo autentizace většího množství uživatelů proti jednomu serveru. To vše Linux umožňuje a mnohdy je využíván pro proskytování těchto služeb i v prostředí heterogenních sítí. Sdílet např. svazky lze přes protokoly SMB/CIFS i s platformou MS Windows prostřednictvím služeb serveru Samba, původní UNIXový způsob sdílení svazků je založen na Network File System (NFS). Moderní metody distribuce většího množství dat reprezentuje na Linuxu Bittorent. Autentizační a konfigurační služby poskytované přes protokol LDAP zprostředkovává server OpenLDAP (na jiných platformách Open Directory nebo Active Directory). Tradiční UNIXový způsob autentizace reprezentuje NIS (ypserv), pro mobilní aplikace je pak určen protokol RADIUS (freeRADIUS).

Databáze

Linux nabízí velké množství databázových strojů a nástrojů. Mezi nejznámější a nepoužívanější patří MySQL a PostgreSQL, v podlední době hojně využíván je též odlehčený engine SQLite. Ke všem velkým databázím v Linuxu existuje přístupová vrstva ODBC případně JDBC pro propojení s Javou.

Komunikace

Přímá komunikace se dnes již odehrává zpravidla na elektronické úrovni. Ani v tomto směru nezůstává Linux pozadu a nabízí množství komunikačních nástrojů. Nejstarším a nejjednodužším z nich je Talk, překonaný mnoha novějšími technologiemi jako např. IRC (Internet Relay Chat), který dnes tvoří konferenční síť kolem celé zeměkoule, nebo Jabber, založený na moderním XML a přinášející mnoho nových služeb a je schopný vytvořit kompletní firemní živé komunikační rozhranní. Mezi důležité prvky součastnosti patří i VoIP (Voice a Video over IP), které jsou v Linuxu reprezentovány službami podle protokolů H323 (OpenH323, Ekiga) a SIP.

Komerční aplikace

Komerční aplikace se od těch výše zmíněných liší tím, že je poskytují a prodávají nejrůznější společnosti. Zpravidla se v takových případech nejedná o software s otevřeným nebo svobodným zdrojovým kódem. Přesto tyto aplikace hrají důležitou roli v serverových aplikacích, neboť jej zpravidla posouvají do kategorie enterprise. Mezi nejznámějšími jmenujme Oracle, Exchange, Lotus Notes a Domino, IBM DB2.
